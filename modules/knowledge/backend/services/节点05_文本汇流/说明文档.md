# 节点⑤ 文本汇流

## 设计意图

统一「按文档/按页」读取三路原始文本，避免 ③/⑥ 各自查 `kb_raw_data`。

## 接口

```python
from modules.knowledge.backend.services.节点05_文本汇流 import 汇流

await 汇流(db, document_id, owner_id, page=1)
# → {text, ocr, vision, vision_desc, ids, meta, status}

await 汇流(db, document_id, owner_id)  # 整文档
# → {pages: {page: ...}, page_count, status}
```

## 语义

| 字段 | round / 来源 |
|---|---|
| text | 1 / text |
| ocr | 2 / ocr |
| vision | 3 content |
| vision_desc | metadata description 优先，否则 round3 content |

## 边界

- **只读**，不写库
- 本轮不改 ③/⑥ 调用方
- 不挂 DAG
