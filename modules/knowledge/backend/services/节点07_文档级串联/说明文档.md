# 节点⑦ 文档级串联

## 职责

单文档：实体抽取 + 当场定类落 type_id + 同文档变体归并。可选摘要/认知索引（默认关，DAG 另有阶段）。

## 目录

```
节点07_文档级串联/
├── 对外接口.py      ★ 唯一对外：串联(db, document_id, owner_id)
├── 实体抽取.py
├── 实体分类落库.py
├── 实体归并.py
├── 文档摘要.py
├── 认知索引.py
└── 说明文档.md
```

## 调用

```python
from modules.knowledge.backend.services.节点07_文档级串联 import 串联

result = await 串联(db, document_id, owner_id)
# {entities_found, relationships_found, typed, pending_review, merged, 样例, status}
```

## 关键规则（写死）

- 提示词：`framework_prompt_templates` id=2（name=knowledge_entity_extraction），只取主体、18 类当场定类
- `confidence < 0.6` → `status=pending_review`，不建图节点、不写证据
- 分类只能用 18 类 type_name → `kb_semantic_types.type_id`
- 归并复用 `semantic_align_service.canonicalize_name` + merge 日志/别名
- 子文件禁止外部 import；graph 只调 `串联`
