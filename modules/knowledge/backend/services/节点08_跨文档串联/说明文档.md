# 节点⑧ 跨文档串联

## 职责

跨文档实体归一 + 主体关系边统计/清自环 + 相似边门禁（砍垃圾「相关」边与弱文档关系）。

## 目录

```
节点08_跨文档串联/
├── 对外接口.py          ★ 跨文档串联(db, document_id, owner_id)
├── 跨文档实体归并.py
├── 主体关系建边.py
├── 相似边门禁.py
└── 说明文档.md
```

## 门禁规则（写死）

- `kb_graph_edges.relation='相关'`：weight < 0.35 删；两端皆噪音删
- `kb_file_relations`：综合分 < 0.35 删；须共享 ≥1 非噪音主体实体
- 目标：别让相似边淹没 拥有/属于/包含

## 接线

`pipeline_stages/relations.py` 调 `跨文档串联`，替换直接 `compute_file_relations`。
