# 节点⑨ 二次融合

## 设计意图

同主体跨文档信息聚合为一个权威视图，并消解（仅标记）冲突。

## 接口

```python
from modules.knowledge.backend.services.节点09_二次融合 import 二次融合
await 二次融合(db, entity_id, owner_id, 提交=True)
```

## 落盘

- 表：`kb_entity_subject_views`
- 冲突：`conflicts_json`，status=open，不自动选赢家
- 不改 raw / fused_text

## 边界

- **未挂 DAG**
- 依赖 ⑦⑧ 质量；脏主体会写硬视图，默认运维点调
