| `lint` | string \| array | 有限并行 | ✅ |

| `finish_task` test | anyOf + list | `run_tests_parallel` | ✅ |

| README MCP 入口 | ZCode + Codex | — | ✅ |

标准 Agent 闭环：

```text
context_load / rg_search / code_*
  → edit
  → select_verify
  → lint(paths)
  → run_tests_parallel(targets, mode=auto)
  → finish_task（同路径并行；可选 auto_select_verify）
```
