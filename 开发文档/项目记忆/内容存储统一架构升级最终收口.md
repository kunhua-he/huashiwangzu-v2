## 最终状态

内容存储统一架构升级（Content IR）已从可用状态收至可合并状态。

### 已完成的核心链路

1. **Content IR 底座** — ir_schema/ir_validator/ir_normalizer/ir_writer 完整，含两层校验（schema + semantic profile）
2. **validate_ir** — viewer 级，返回结构化 errors（path/code/message/expected/actual）
3. **write_ir** — editor 级，写前二次校验，非法 IR 返回 CONTENT_IR_VALIDATION_FAILED + data.errors
4. **compile** — viewer 级，临时编译不创建 file record，路径安全检查
5. **store_analysis_resource** — viewer 级，VLM/分析结果回填，内部 check_file_access
6. **excel-engine:compile_xlsx** — viewer 级，不存在 workbook 返回失败
7. **Spreadsheet IR** — headers 写入首行，支持 start_cell
8. **下载链路** — ContentPackage→compile, Excel→compile_xlsx, spreadsheet 跳过 ContentPackage compile
9. **Package 创建** — 无 source_file_id 时不复用孤儿 package
10. **权限链** — Resource 读取继承 file share（ResourceRef→ContentPackage→source_file_id）
11. **System principal** — resolve_caller_user_id 接受 system:* 返回 0，write_ir 拒绝 owner_id=0
12. **Agent 敏感阻断** — SYSTEM_HARD_BLOCKED_ACTIONS + user_id=0 硬阻断物理文件生成
