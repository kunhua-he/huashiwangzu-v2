# MCP 使用反馈

## 任务

验收并提交资产生命周期门禁实现，写入资产债务清偿主闭环二期执行信

## 顺畅度

- 评分：4/5
- 体感：整体顺畅，worktree_guard/lint/run_test/release_gate/Bash 推送链路可用。

## 本次用到的工具

worktree_guard, run_test, lint, probe, release_gate, finish_task, memory_write, mcp_feedback, Bash

## 卡点 / 不顺手的地方

finish_task 对混合 backend/tests 与 repo-root 测试目标一次性聚合执行时，cwd/env 与 run_test 分目标执行不同，导致 JWT_SECRET is empty；分目标 run_test 实际通过。

## 缺少的工具 / 能力

希望 finish_task 支持按目标自动分组运行测试，backend/tests 在 backend cwd，repo-root/dev_toolkit/modules 走 repo cwd，避免混跑环境问题。

## 升级建议

finish_task 可接收 already_verified JSON 并只记录不重跑；或提供 per-target cwd/env 归一化。

## 建议移除或合并的工具

无
