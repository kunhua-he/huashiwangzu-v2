# 执行信：桌面窗口动画与鼠标磁吸分屏

## 目标

增强桌面窗口体验：窗口打开/关闭/最小化有轻量过渡，拖拽到屏幕边缘支持鼠标磁吸分屏提示与落位。

## 修改边界

只允许：

```text
frontend/src/desktop/window-manager/
frontend/src/desktop/shell/
frontend/src/desktop/taskbar/ 与窗口最小化/恢复直接相关文件
frontend/src/styles/ 或已有窗口样式文件
frontend/tests/
开发文档/项目记忆/
```

禁止：

```text
backend/
dev_toolkit/
modules/
frontend/src/shared/api/
frontend/src/shared/files/
frontend/src/desktop/launcher/
```

## 必做

1. 窗口打开/关闭/最小化/恢复有轻量动画，不影响性能。
2. 拖拽窗口到左/右/上边缘时显示磁吸预览。
3. 松手后落位：左半屏、右半屏、最大化。
4. 多窗口状态与现有 window manager 兼容。
5. 不破坏现有窗口尺寸、最小宽高、singleton 行为。

## 验收

```bash
cd frontend && npm run build
```

目标文件扫描：

```text
无 any/as any/@ts-ignore/@ts-expect-error
```
