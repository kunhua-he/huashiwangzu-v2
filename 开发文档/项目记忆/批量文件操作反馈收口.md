# 改了什么
- 仅修改 `frontend/src/shared/files/use-file-operations.ts`。
- 新增 `BatchOperationResult` / `BatchOperationError` 结构，`pasteToFolder` 对 copy/move/paste 批量路径逐项累计 `successCount`、`failCount` 和失败明细。
- `deleteEntry` 也在本地形成并返回单项 `BatchOperationResult`，失败不再只给笼统警告。
- UI 提示区分：全部成功（多项显示完成数）、部分成功（完成 N 个，失败 M 个）、全部失败；失败项摘要进 toast，完整失败数组进 `console.warn`。
