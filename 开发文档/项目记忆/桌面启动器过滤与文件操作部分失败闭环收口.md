# 改了什么
- 启动器普通应用列表只展示 showInLauncher 且可直接打开窗口的 app。
- 新增 app-visibility 判断，background-service/background 不作为普通启动器应用。
- 命令搜索新增 background-capability 类型，UI 标注“后台能力/不可直接打开”。
- 后台能力/不可直接打开 app 或 action 点击时提示“该能力是后台服务，不能直接打开窗口”。
- openWindow 返回 null 的主要调用侧补用户可见反馈，并用 getOpenWindowFailureMessage 区分缺应用、停用、后台服务和其他失败。
- use-file-operations 新增 BatchOperationResult/BatchOperationError，paste copy/move 逐项统计成功/失败，失败详情进入 toast 摘要和 console.warn；deleteEntry 也返回单项结构化结果。
- 新增 frontend/tests/desktop-launcher-fileops.spec.mjs，覆盖后台能力搜索点击提示和批量粘贴部分失败计数。
