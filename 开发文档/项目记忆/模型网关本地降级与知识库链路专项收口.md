# 改了什么

- 模型网关按 `backend/data/config/models.json` 的 fallback chain 降级：显式云端 profile 失败也可降级，本地优先顺序调整为 llama.cpp (`gemma-4`) -> Ollama -> 云端备用；协议错误不降级，避免掩盖 tool-call 序列 bug。
- 知识库 `register_document` 收口同内容三态去重：已索引内容复用；正在入库复用同一个 in-flight 文档；零 chunk 的终态孤儿不阻塞新入库。
- 新增 `knowledge:get_ingest_status` 和 `/documents/{id}/ingest-status`，统一返回 queue + parse/vector/raw/fusion + source lifecycle 状态；源文件删除/缺失稳定表达为 `source_unavailable`。
- pipeline 阶段语义从“有无 error”升级为 `done/degraded/failed/skipped`，raw/fusion 全空不再伪装完成；degraded 任务在 worker 层保持 completed，但结果可见。
- search/chunk 读取过滤 deleted document、deleted/missing source file；修复 pgvector 返回 `numpy.ndarray` 时向量检索静默空的问题。
